# COMP2000 Coursework 2 Repo
This repo contains the Android Studio project for my COMP2000 submission. This project is a robust, role based restaurant app working for guests, staff and managers.

## Features
### Guest
- View menu
- Creating booking
- Edit bookings
- Edit and view own profile
- Contact restaurant

### Staff
- View all bookings
- View today's bookings
- Edit all bookings
- View menu
- Edit and view own profile

### Manager
- Edit menu
- Create menu items
- Delete menu items
- Edit and view all bookings
- Staff management
  - Edit staff details
  - Add new staff members
  - Remove staff accounts
- Edit and view own proifle

## How to use
1. Build app in android studio
2. Install on device
3. Connect to PlymUni Vpn on device
4. Start app and register account
